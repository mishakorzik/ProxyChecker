## ProxyChecker
**This is a very powerful and fast proxy checker, can check a bunch of proxies in seconds!**

### Requirements

* python3
* requests

be sure to install python and install the requests module via pip

---

### Manual installing
```bash
git clone https://github.com/mishakorzik/ProxyChecker
cd ProxyChecker
python3 checker.py
```
ProxyChecker can check a large number of proxies with great speed, supports protocols: socks5, socks4 http
